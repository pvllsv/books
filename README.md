# Landing-страница книги

## Описание
Этот проект представляет собой адаптивную посадочную страницу. Он включает разделы с информацией об авторе, главах книги, отзывах читателей, статьях и формой подписки.

## Используемые технологии
- HTML5
- CSS3
- JavaScript

## Просмотр онлайн
Вы можете посмотреть результат здесь: [https://pvllsv.github.io/books/](https://pvllsv.github.io/books/)

## Как запустить
1. Клонируйте этот репозиторий на вашу локальную машину.
2. Перейдите в директорию `books`.
3. Откройте файл `index.html` в вашем веб-браузере.
